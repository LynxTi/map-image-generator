# Parchment Map Generators

Standalone project with three generators for vintage "parchment" maps:
1. Vector SVG (`src/variant1-vector.js`)
2. Real Maps via MapLibre headless (`src/variant2-real.js`)
3. Hybrid (clean vector + raster aging) (`src/variant3-hybrid.js`)

## Usage
Run the demo to generate 30 WebP maps (10 locations x 3 variants):
```bash
npm run demo
```
